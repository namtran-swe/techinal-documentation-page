Introduction

This would be the introduction to the webpage.

On this page you will learn basics of coding, specifically about HTML and CSS.

HTML Section

This section will talk about HTML.

To start off HTML code you always need to declare
!DOCTYPE html
and after that declare your HTML tag like so -
html lang="en"

There are many other tags that you will use in HTML such as div, img, and select.

CSS Section

This section will talk about CSS.

CSS is used to style HTML elements and make your webpage spicy.

An important part of CSS is the box model. The box model consists of:

  • The content
  • Padding
  • Border
  • Margin

JS Section

This section will briefly touch upon JavaScript

If you think JavaScript and Java are similar then you better think again. JavaScript is used to add functionality to the elements you have implemented and styled with HTML and CSS.

Once you have the basics of JavaScript down you will be able to utilize some of its libraries like:

  • Angular
  • React
  • Vue.js

Conclusion

Let's wrap it up
